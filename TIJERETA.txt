adicionales:
manual de usuario
presentacion (<12 diap, empezar con motivo de proy, capturas; 15')
presentacion 2 (demostracion gral y luego simulacion, solicitar participacion prof; 25')
diagrama de base de datos

estadísticas de uso, tendencias, promedios
historial:
registro turnos, historial turnos cliente, filtrado de turnos solicitados por clientes
api whatsapp:
envío de mensaje de confirmación al whatsapp del cliente usando número almacenado del mismo
busqueda:
barra de búsqueda universal a toda la app o individual para cada página, con la posibildiad
de seleccionar y editar desde la lista de resultados.
